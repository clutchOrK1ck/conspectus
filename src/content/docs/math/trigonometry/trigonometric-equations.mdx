## Elementary trigonometric equations

### \` sin(x) = a \`

The equation $ sin x = a $ has solutions if $ a in [-1; 1] $

Then,

$$ 
    x = {(arcsin a + 2 pi n),(pi - arcsin a + 2 pi n):}\ \ ", where"\ n in Z
$$

#### Special cases

| $ sin x = ... $ | $ x = ... $ |
| ---- | ---- |
| $ 0 $ | $ pi n,\ n in Z $  |
| $ 1 $ | $ pi/2 + 2 pi n,\ n in Z $ |
| $ -1 $ | $ pi/2 + 2 pi n,\ n in Z $ |

### \` cos(x) = a \`

The equation $ cos x = a $ has solutions if $ a in [-1; 1] $

Then,

$$ 
    x = +- arccos a + 2 pi n\ \ ", where"\ n in Z
$$

#### Special cases

| $ cos x = ... $ | $ x = ... $ |
| ---- | ---- |
| $ 0 $ | $ pi/2 + pi n,\ n in Z $  |
| $ 1 $ | $ 2 pi n,\ n in Z $ |
| $ -1 $ | $ pi + 2 pi n,\ n in Z $ |

## Solution methods

### Variable substitution

Trigonometric equations can be reduced to algebraic equations through variable substitution. For example, the equation $ 4 cos^2 2x + 16 sin 2x - 11 = 0 $ is reduced to a plain quadratic equation by rearranging and substituing $ sin2x -> t $

#### Example

Solve trigonometric equation: 

$$ 
4 cos^2 2x + 16 sin 2x - 11 = 0 
$$

<Solution>

$$nl
    4(1 - sin^2 2x) + 16 sin 2x - 11 = 0
    4 - 4 sin^2 2x + 16 sin 2x - 11 = 0
$$

Perform substitution:  $ sin2x -> t $, where $ |t| <= 1 $

$$nl
    -4 t^2 + 16 t - 7 = 0
    4t^2 - 16t + 7 = 0
    t_(1,2) = [\{:(7/2),(1/2):} " where " 7/2 " does not satisfy " |t| <= 1
$$

Substitute back: $ sin 2x -> 1/2 $

$$nl
    2x = {(arcsin (1/2) + 2 pi n), (pi - arcsin (1/2) + 2 pi n):},\ n in Z
    x = {(pi/12 + pi n), ((5pi)/12 + pi n):},\ n in Z
$$
</Solution>

#### Example

Solve equation:

$$
    2 tan x + 3 cot x = 5
$$

<Solution>
Domain: $ x != 0 + pi n, x != pi/2 + pn, n in Z $

$$nl
    cot x -> 1 / (tan x)
    2 tan x + 3 / (tan x) = 5
    (2 tan^2 x - 5 tan x + 3)/(tan x) = 0
$$

Note that $ tan x != 0 => x != 0 $ is already covered by the domain of $ x $

$$nl
    2 tan^2 x - 5 tan x + 3 = 0
    tan x -> u
    2 u^2 - 5 u + 3 = 0
    u_(1,2) = [{:(3/2),(1):}
$$

Then, substituting back:

$$nl
    [{:(tan x = 3/2), (tan x = 1):}
    x = [{:(arctan (3/2) + pi n), (arctan 1 + pi n):}, n in Z
    x = [{:(arctan (3/2) + pi n), (pi/4 + pi n):}, n in Z
$$
</Solution>

### Reducing to \` f(x)*g(x) = 0 \`

If possible to rewrite as $ f(x) * g(x) = 0 $, then it follows that $ f(x) = 0 $ and/or $ g(x) = 0 $. Both equations are then solved separately.

#### Example

Solve:

$$
    sin x tan x - sin x = 1 - tan x
$$

<Solution>

$$nl
    sin x (tan x - 1) = - (tan x - 1)
    (tan x - 1) (sin x + 1) = 0
$$

Domain: $ x != pi / 2 + pi k, k in Z $ because of $ tan(x) $

Then,

$$nl
    tan x = 1
    x = arctan 1 + pi n, n in Z
    x = pi / 4 + pi n, n in Z
$$

and

$$nl
    sin x = -1
    x = pi / 2 + 2 pi n, n in Z
$$

...which is not in the set of permissible values.

Finally,

$$nl
    x = pi / 4 + pi n, n in Z
$$

</Solution>

### Reducing to \` f(x)/g(x) = 0 \`

If possible to rewrite as $ f(x) / g(x) = 0 $, then, given that $ g(x) != 0 $, the whole equation is reduced to $ f(x) = 0 $ by multiplying both sides by $ g(x) $

#### Example

Solve:

$$
    cot x - tan x = (cos x - sin x)/(1/2 sin 2x)
$$

<Solution>

$$nl
    cos x / (sin x) - sin x / (cos x) = (cos x - sin x) / (1/2 sin 2x)
    (cos^2 x - sin^2 x)/(sin x cos x) = (cos x - sin x) / (1/2 sin 2x)
$$

Note that:
$$
sin 2x = 2 sin x cos x " " => " " sin x cos x = 1/2 sin 2x 
$$

Then,
$$nl
    (cos^2 x - sin^2 x)/(1/2 sin 2x) - (cos x - sin x)/(1/2 sin 2x) = 0
    ((cos^2 x - sin^2 x) - (cos x - sin x)) / (1/2 sin 2x) = 0
$$

Define the set of permissible values:

$$nl
    sin 2x != 0
    2x != pi n, n in Z
    x != pi/2 n, n in Z
$$

Then, we need to solve

$$nl
    (cos^2 x - sin^2 x) - (cos x - sin x) = 0
    (cos x - sin x)(cos x + sin x) - (cos x - sin x ) = 0
    (cos x - sin x)(cos x + sin x - 1) = 0
$$

Solving $ cos x - sin x = 0 $:

$$nl
    cos x - sin x = 0 | div cos x
    NB: cos x != 0 => x != pi/2 + pi n, n in Z "(already covered)"
    1 - tan(x) = 0
    tan(x) = 1
    x = arctan(1) + pi n, n in Z
    x = pi/4 + pi n, n in Z
$$

Solving $ cos x + sin x - 1 = 0 $:

$$nl
    cos x + sin x = 1
    cos^2 x + 2 sin x cos x + sin^2 x = 1^2
    cos^2 x + 2 sin x cos x + sin^2 x = sin^2 x + cos^2 x
    2 sin x cos x = 0
    sin 2x = 0
    2x = pi n, n in Z
    x = pi/2 n, n in Z
$$

Since the last solution is not in the set of permissible values,

$$
    x = pi/4 + pi n, n in Z
$$
</Solution>

### Homogeneous trigonometric equations

Homogeneous trigonometric equations are defined as [homogeneous polynomials](/math/concepts/#homogeneous-polynomials) of $ cos(x), sin(x) $ equal to 0:

$$
    sum_(i = 0 .. n) a_i * sin^(n-i) x * cos^i x = 0
$$

... or, if expanded:

$$width = null, overflow = scroll
    a_0 sin^n(x) + a_1 sin^(n-1)(x) cos(x) + a_2 sin^(n-2)(x) cos^2(x) + ... + a_n cos^n x = 0
$$

where $ a_i in R $

These equations are solved by dividing by $ sin^n x $ or $ cos^n x $

#### Example

Solve:

$$
    3 sin^2 x - 5 sin x cos x - 8 cos^2 x = 0
$$

<Solution>
    $$nl, justify = left
        3 sin^2 x - 5 sin x cos x - 8 cos^2 x = 0 | div cos^2 x, cos x != 0, x != pi/2 + pi n
        3 tan^2 x - 5 tan x - 8 = 0
        tan x -> t, x != pi/2 + pi n
        3 t^2 - 5 t - 8 = 0
        D = 25 + 96 = 121
        sqrt D = +- 11
        t_(1,2) = (5 +- 11) / 6
        t = [\{:(8/3),(-1):}
        => 
        1) tan x = 8/3
        x = arctan (8/3) + pi n, n in Z
        2) tan x = -1
        x = arctan (-1) + pi n = -pi/4 + pi n, n in Z
    $$
</Solution>

### Equations reducible to homogeneous

Equations of the form:

$$
    a_0 sin^n x + ... + a_n cos^n x = d
$$

can *sometimes* be reduced to normal homogeneous equations by rewriting $ d $ as $ d(sin^2 x + cos^2 x) $. Given a passing form of $ d $, subtracting $ d (sin^2 x + cos^2 x) $ from both sides will yield a normal homogeneous equation.

#### Example

Solve:

$$
    2 sin^3 x = cos x
$$

<Solution>
    $ 2 sin^3 x = cos x $ *could* be a homogeneous equation of degree 3. This can be indeed achieved by rewriting as:

    $$nl
        2 sin^3 x = cos x (sin^2 x + cos^2 x)
        2 sin^3 x - cos x sin^2 x - cos^3 x = 0
    $$

    ...which is a homogeneous equation, solved in the usual way.

    $$nl
        2 sin^3 x - cos x sin^2 x - cos^3 x = 0 | div cos^3 x
        2 tan^3 x - tan^2 x - 1 = 0
        tan x -> t
        2 t^3 - t^2 - 1 = 0
        t^3 + t^3 - t^2 - 1 = 0
        t^3 - 1 + t^3 - t^2 = 0
        (t - 1)(t^2 + t + 1) + t^2(t - 1) = 0
        (t - 1)(2t^2 + t + 1) = 0
    $$

    ... and so on.
</Solution>

### Form \` R(sin(nx), cos(kx), tan(mx), cot(lx)) = 0 \`

Equations of form $ R(sin(nx), cos(kx), tan(mx), cot(lx)) = 0 $ where $ R $ is a [rational function](/math/concepts/#rational-functions) are solvable through universal trigonometric substitution $ t = tan(x/2) $. Then, using [these trig formulas](/math/trigonometry/main-trigonometric-formulas/#tan-alpha-over-2), it follows that:

$$nl
    sin x = (2t) / (1 + t^2)
    cos x = (1 - t^2) / (1 + t^2)
    tan x = (2t) / (1 - t^2)
$$

#### Example

Solve:

$$
    5 sin 2x - 5 cos 2x = tan x + 5
$$

<Solution>
    The equation conforms to $ R(sin(nx), cos(kx), tan(mx), cot(lx)) = 0 $ because it can be rewritten as

    $$
        5 sin 2x - 5 cos 2x - tan x - 5 = 0
    $$

    In this case, we can think of the denominator as simply $ sin^0 (2x) $ or similar.

    Perform substitution: $ tan x -> t $. Then,

    $$nl
        sin 2x = (2t) / (1 + t^2)
        cos 2x = (1 - t^2) / (1 + t^2)
        5 * (2t) / (1 + t^2) - 5 * (1 - t^2)/(1 + t^2) - t - 5 = 0
    $$

    Note that $ x != pi / 2 + pi n, n in Z $

    $$nl
        (10t - 5 + 5t^2)/(1 + t^2) - ((t + 5)(1 + t^2))/(1 + t^2) = 0
        ...
        (-t^3 + 9t - 10)/(1+t^2) = 0
        [(t^3 - 9t + 10 = 0), (1 + t^2 != 0):}
        t = [(2), (-1 + sqrt(6)), (-1 - sqrt(6)):} 
        "substitute back" =>
        tan(x) = [(2), (-1 + sqrt(6)), (-1 - sqrt(6)):}
    $$

    These equations are elementary and solved correspondingly.
    
</Solution>

### Homogeneous equations of 1st degree

Equations of shape

$$
    a * sin(mx) +- b * cos(mx) = c
$$

where $ a, b, c $ are constants.

Methods: universal trigonometric substitution or the method of auxilliary angle.

Method of auxilliary angle: divide both sides by $ sqrt(a^2 + b^2) $. Then:

$$nl
    a / sqrt(a^2 + b^2) = sin(alpha)
    b / sqrt(a^2 + b^2) = cos(alpha)
$$

or 

$$nl
    a / sqrt(a^2 + b^2) = cos(theta)
    b / sqrt(a^2 + b^2) = sin(theta)
$$

for some angles $ alpha, theta $ in a right triangle. Then solve using [identities](/math/trigonometry/main-trigonometric-formulas#functions-of-sum) of trigonometric functions of sum of arguments

<Solution promptShow = "Show explanation" promptHide = "Hide explanation">
    Assume a right triangle whose legs are equal to $ a, b $ from the equation. Then the hypotenuse of this triangle is $ sqrt(a^2 + b^2) $. Then, when we divide both sides of the equation by $ sqrt(a^2 + b^2) $, we receive:

    $$
        a / sqrt(a^2 + b^2) sin(mx) +- b / sqrt(a^2 + b^2) cos(mx) = c / sqrt(a^2 + b^2)
    $$

    If we arbitrarily choose an acute angle $ alpha $ (resp. $ theta $) in this right triangle, $ a $ is an adjacent (resp. opposite) side to it, with $ a / sqrt(a^2 + b^2) = cos(alpha) $ (resp. $ sin(theta) $), and $ b $ is opposite (resp. adjacent) side to it, with $ b / sqrt(a^2 + b^2) = sin(alpha) $ (resp. $ cos(theta) $).

    Then we can write this equation as

    $$nl
        sin(alpha) sin(mx) +- cos(alpha) cos(mx) = c / sqrt(a^2 + b^2)
        or
        cos(theta) sin(mx) +- sin(theta) cos(mx) = c / sqrt(a^2 + b^2)
    $$

    which can be then solved using trigonometric identities to rewrite the left side as a trigonometric function of sum of arguments.
</Solution>

#### Example

Solve: 

$$
    sqrt(3) sin(3x) - cos(3x) = 1
$$

<Solution>
    Start by dividing both sides by the hypotenuse of a right triangle with legs $ sqrt(3), 1 $:

    $$nl
        sqrt( sqrt(3)^2 + 1^2 ) = sqrt( 4 ) = 2
        sqrt(3) / 2 sin(3x) - 1 / 2 cos(3x) = 1 / 2
    $$

    It does not matter if we assign $ sqrt(3)/2 = cos(alpha) $ or $ sqrt(3)/2 = sin(alpha) $ for some angle $ alpha $, see both solutions:

    <Tabs>
        <TabItem label="` sqrt(3)/2 = cos(alpha) `">
            $$nl
                cos(alpha) sin(3x) - sin(alpha) cos(3x) = 1/2
                sin(3x - alpha) = 1/2
                3x - alpha = {(arcsin (1/2) + 2 pi n), (pi - arcsin (1/2) + 2 pi n):}
                3x = {(arcsin (1/2) + 2 pi n + arccos(sqrt(3)/2)), (pi - arcsin (1/2) + 2 pi n + arccos(sqrt(3)/2)):}
                3x = {(pi/3 + 2 pi n), (pi + 2 pi n):}
                x = {(pi/9 + 2/3 pi n), (pi/3 + 2/3 pi n):}
            $$ 
        </TabItem>
        <TabItem label="` sqrt(3)/2 = sin(alpha) `">
            $$nl
                sin(alpha) sin(3x) - cos(alpha) cos(3x) = 1/2
                cos(alpha + 3x) = -1/2
                alpha + 3x = +- arccos(-1/2) + 2 pi n, n in Z
                3x = +- arccos(-1/2) - arcsin(sqrt(3)/2) + 2 pi n, n in Z
                3x = +- (2pi)/3 - pi/3 + 2 pi n, n in Z
                x = +- (2pi)/9 - pi/9 + 2/3 pi n, n in Z
                x = [(pi/9 + 2/3 pi n), (-pi/3 + 2/3 pi n):}, n in Z
            $$

            Note that $ -pi/3 + 2/3 pi n $ is essentially the same sequence as $ pi/3 + 2/3 pi n $
        </TabItem>
    </Tabs>
</Solution>

### Shape \`R(sinx +- cosx; sinx*cosx)\`

Method: substitution $ t = sinx +- cosx $

Then:

$$nl
    t^2 = sin^2(x) +- 2 sinx cosx + cos^2(x)
    sinx cosx = +- (t^2 - 1)/2
$$

#### Example

Solve:

$$
    sinx + cosx - 2sqrt(2) sinx cos x = 0
$$

<Solution>
    $$nl
        sin x + cos x -> t
        => t^2 = sin^2 + 2 sin x cos x + cos^2
        => sin x cox = (t^2 - 1)/2
        t - 2sqrt(2) (t^2 - 1)/2 = 0
        sqrt(2)t^2 - t - sqrt(2) = 0
        t^2 - sqrt(2) / 2 t - 1 = 0
        ...
        t = [(sqrt(2)), (-sqrt(2)/2):}
        "substitute back" =>
        [(sinx + cosx = sqrt(2)), (sinx + cosx = -sqrt(2)/2):}
    $$

    The latter equations are solved using one of the methods described above.
</Solution>
