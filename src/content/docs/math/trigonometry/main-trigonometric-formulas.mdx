---
title: Trigonometric identities
---

### Basic identities

$$nl, justify = left

    1.\ sin^2 alpha + cos^2 alpha = 1
    2.\ tan alpha = sin alpha / cos alpha
    3.\ cot alpha = cos alpha / sin alpha
    4.\ tan alpha cot alpha = 1
    5.\ tan^2 alpha + 1 = 1 / cos^2 alpha
    6.\ cot^2 alpha + 1 = 1 / sin^2 alpha

$$

### Functions of \` alpha / 2 \`

$$nl, justify = left

    1.\ 2 sin^2 (alpha/2) = 1 - cos alpha
    2.\ 2 cos^2 (alpha/2) = 1 + cos alpha
    3.\ tan (alpha/2) = (1 - cos alpha)/sin alpha = sin alpha/(1 + cos alpha)

$$

### Functions of \` 2 alpha \`

$$nl, justify = left

    1.\ sin 2 alpha = 2 sin alpha cos alpha
    2.\ cos 2 alpha = cos^2 alpha - sin^2 alpha = 1 - 2 sin^2 alpha = 2 cos^2 alpha - 1
    3.\ tan 2 alpha = (2 tan alpha) / (1 - tan^2 alpha)
    4.\ cot 2 alpha = (cot^2alpha - 1) / (2 cot alpha)

$$

### Functions of \` 3 alpha \`

$$nl, justify = left

    1.\ sin 3 alpha = 3 sin alpha - 4 sin^3 alpha
    2.\ cos 3 alpha = 4 cos^3 alpha - 3 cos alpha
    3.\ tan 3 alpha = (3 tan alpha - tan^3 alpha) / (1 - 3 tan^2 alpha)
    4.\ cot 3 alpha = (3 cot alpha - cot^3 alpha) / (1 - 3 cot^2 alpha)

$$

### Functions of sum

$$nl, justify = left

    1.\ sin(alpha + beta) = sin alpha cos beta + cos alpha sin beta
    2.\ sin(alpha - beta) = sin alpha cos beta - cos alpha sin beta
    3.\ cos(alpha + beta) = cos alpha cos beta - sin alpha sin beta
    4.\ cos(alpha - beta) = cos alpha cos beta + sin alpha sin beta
    5.\ tan(alpha + beta) = (tan alpha + tan beta) / (1 - tan alpha tan beta)
    6.\ tan(alpha - beta) = (tan alpha - tan beta) / (1 + tan alpha tan beta)
    7.\ cot(alpha + beta) = (cot alpha cot beta - 1) / (cot alpha + cot beta)
    8.\ cot(alpha - beta) = - (1 + cot alpha cot beta) / (cot alpha - cot beta)

$$

### Sum of functions

$$nl, justify = left

    1.\ sin alpha + sin beta = 2 sin ((alpha + beta) / 2) cos ((alpha - beta) / 2)
    2.\ sin alpha - sin beta = 2 cos((alpha + beta)/2) sin((alpha - beta)/2)
    3.\ cos alpha + cos beta = 2 cos((alpha + beta)/2) cos((alpha - beta)/2)
    4.\ cos alpha - cos beta = - 2 sin((alpha + beta)/2) sin ((alpha - beta)/2)
    5.\ tan alpha + tan beta = sin(alpha + beta)/(cos alpha cos beta)
    6.\ tan alpha - tan beta = sin(alpha - beta)/(cos alpha cos beta)
    7.\ cot alpha + cot beta = sin(alpha + beta)/(sin alpha sin beta)
    8.\ cot alpha - cot beta = - sin(alpha - beta)/(sin alpha sin beta)

$$

### Product of functions

$$nl, justify = left

    1.\ sin alpha sin beta = 1/2\ [ cos(alpha - beta) - cos(alpha + beta)]
    2.\ cos alpha cos beta = 1/2\ [ cos(alpha - beta) + cos(alpha + beta)]
    3.\ sin alpha cos beta = 1/2\ [ sin(alpha - beta) + sin(alpha + beta)]

$$

### Expressing through \` tan(alpha / 2) \` {#tan-alpha-over-2}

$$nl, justify = left

    1.\ sin alpha = (2 tan(alpha/2) ) / ( 1 + tan^2(alpha/2))
    2.\ cos alpha = (1 - tan^2(alpha/2) ) / ( 1+ tan^2(alpha/2))
    3.\ tan alpha = (2 tan(alpha/2)) / (1 - tan^2(alpha/2))
    4.\ cot alpha = (1 - tan^2(alpha/2)) / (2tan(alpha/2))

$$
